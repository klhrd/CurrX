# **CurrX 貨幣計算機 - 首頁元件規格 (HTML/JS 版)**

本文件定義 CurrX 應用程式主畫面的視覺元件與互動邏輯，特別針對 HTML 實作、資料持久化與視覺美感進行優化。

## **1. 頂部導航與狀態欄 (Top Header)**

- **應用程式名稱：** 左上角顯示「CurrX」。  
- **數據更新狀態：** 中央顯示「Last refresh: [時間] ago」。  
    - **LocalStorage 緩存邏輯：** 匯率數據將暫存於 localStorage 中。每次開啟 App 時先讀取快取，若快取超過預設時間（如 1 小時）或點擊「🔄」，才向 API 請求新數據。  
- **功能按鈕：**  
    - **重新整理 (🔄)：** 點擊時強制刷新 localStorage 內的匯率並更新顯示時間。  
    - **更多選單 (⋮)：** 進入設定頁。

## **2. 匯率轉換顯示區 (Currency Display)**

- **幣別切換按鈕 (Currency Selectors)：**  
    - 顯示 USD 與 TWD（或使用者自訂幣別）。  
    - 中間夾帶 **雙向箭頭 (⇄)**：執行「一鍵反向換算」邏輯，單純交換幣值不影響運算式。  
- **輸入與結果顯示：**  
    - **算式輸入欄 (Expression Field)：** 顯示運算過程，字體較小，顏色稍淡。  
    - **數值結果欄 (Result Field)：** 顯示主幣別金額與換算後金額，字體加大並加粗。

## **3. 智慧計算按鍵區 (Input Pad)**

### **A. 視覺規範 (Google Fonts 應用)**

- **字體選擇：** 全站使用 **Google Fonts**。  
    - 數字與運算符號建議使用 Roboto 或 Poppins（具備良好的幾何平衡感）。  
    - 介面標籤建議使用 Noto Sans TC 以確保中文顯示美觀。  
- **按鈕外觀：** 
    - 使用 **Google Fonts Icons**。
    - 使用 CSS 變數管理顏色，支援「深色/淺色模式」。  
    - 圓角設計（Border-radius: 12px+），並加入微影音效果。

### **B. 功能控制類**

- **AC (All Clear)：** 清除輸入。  
- **退格鍵 (⌫)：** 使用圖示（Icon）代替文字。  
- **括號 ( )、百分比 (%)、加減乘除 (+, -, ×, ÷)：** 運算符號。  
- **等號 (=)：** 點擊後計算結果並將此筆交易存入 localStorage 的歷史紀錄陣列中。

## **4. 實作細節與優化 (Small Details)**

- **觸覺回饋 (Haptic Feedback)：** 若在手機瀏覽器開啟，按鈕點擊時可加入短暫的震動 API (window.navigator.vibrate)。  
- **防錯機制：** * 輸入非法算式時（如 5++8），結果欄顯示「格式錯誤」而非程式報錯。  
    - LocalStorage 讀取失敗時，提供預設的「最後已知匯率」作為備案。  
- **響應式文字：** 數字過長時，字體大小應自動縮小（利用 CSS clamp() 或 JS 動態偵測長度），避免破版。  
- **輸入動畫：** 數字出現時帶有輕微的淡入或平移效果。

# **CurrX 貨幣計算機 - 選單功能詳細規格**

本文件定義了 CurrX 側邊選單（或設定頁面）的六大核心功能邏輯與 UI 呈現建議。

## **1. 顯示設定 (Display Settings)**

### **🟢 小數點位數 (Decimal Precision)**

- **功能描述：** 調整計算結果與匯率顯示的精細度。  
- **選項：** 0 位 / 2 位 (預設) / 3 位 / 4 位。  
- **互動邏輯：** 切換後，主畫面上的 USD 與 TWD 欄位數值應即時連動更新縮位或補零。

### **🟢 深色/淺色模式 (Theme Appearance)**

- **功能描述：** 切換應用程式的視覺主題色調。  
- **選項：** 淺色 (Light) / 深色 (Dark) / 跟隨系統。  
- **設計建議：** 深色模式下應使用高對比度的文字（如純白或淺灰），確保計算按鈕與算式清晰可見。

## **2. 數據與計算邏輯 (Data & Logic)**

### **🟢 匯率來源選擇 (Exchange Rate Source)**

- **功能描述：** 選擇計算換算結果時所依據的數據庫 (皆是免費不須的API) 。  
- **選項：**  
  1. **ExchangeRate：**  
  2. **Fawazahmed0：**  
  3. **Frankfurter**  
- **介面顯示：** 在主畫面的「Last Refresh」旁應顯示刷新按鈕。

## **3. 工具與歷史 (Tools & History)**

### **🟢 歷史紀錄回溯 (History Log)**

- **功能描述：** 查看過去的計算軌跡。  
- **紀錄內容：** * 完整的算式（例如：(5+8)x2+13%）。  
    - 當下的匯率來源與數值。  
    - 換算後的兩國貨幣結果。  
- **快捷操作：** 點擊任一紀錄可將該數值重新帶入主計算機介面。

### **🟢 複製與分享 (Copy & Share)**

- **功能描述：** 快速將當前的換算結果匯出。  
- **格式建議：** > [CurrX 換算紀錄]輸入：(5+8)x2 + 13% = 29.38 USD  
  結果：****** TWD  
  匯率來源：台灣銀行 (1 USD = ******** TWD)  
- **操作：** 點擊選單中的「分享」後彈出系統分享清單。

## **UI 佈局建議 (選單結構)**

1. **紀錄中心** (歷史紀錄、複製分享)  
2. **偏好設定** (小數點位數、反向邏輯、主題模式)  
3. **數據服務** (匯率來源、手動更新)
